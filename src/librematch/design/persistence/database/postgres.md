## Database migration

We version control our database schema with a lightweight, framework-agnostic database migration tool called [dbmate](https://github.com/amacneil/dbmate).

The database schema and migrations are open-source and can be found [here](https://github.com/librematch/librematch-database/). This repository acts like a **single source of truth (SSOT)** for the DB schema.
