# Advertisement Endpoints (Community)

Public endpoints for searching game lobbies without authentication.

> **Note:** This is an unauthenticated public endpoint. For authenticated lobby operations (create, join, host), see [/game/advertisement](../../game/advertisement/).

## Endpoints

| Endpoint                                      | Method | Description                    |
| --------------------------------------------- | ------ | ------------------------------ |
| [findAdvertisements](./findadvertisements.md) | GET    | Search for active game lobbies |
