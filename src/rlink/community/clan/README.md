# Clan Endpoints (Community)

Public endpoints for searching clans and viewing clan information without authentication.

> **Note:** These are unauthenticated public endpoints. For authenticated clan operations (create, join, manage), see [/game/clan](../../game/clan/).

## Endpoints

| Endpoint                                | Method | Description                                     |
| --------------------------------------- | ------ | ----------------------------------------------- |
| [find](./find.md)                       | GET    | Search for clans by name or tag                 |
| [getClanInfoFull](./getclaninfofull.md) | GET    | Get detailed clan information including members |
