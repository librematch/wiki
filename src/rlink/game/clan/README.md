# Clan Endpoints

Endpoints for clan/guild management including creation, membership, and settings.

> **Authentication:** All endpoints in this section require Steam authentication.

## Endpoints

| Endpoint                                | Method | Description                         |
| --------------------------------------- | ------ | ----------------------------------- |
| [apply](./apply.md)                     | POST   | Apply to join a clan                |
| [create](./create.md)                   | POST   | Create a new clan                   |
| [disband](./disband.md)                 | POST   | Disband/delete a clan               |
| [find](./find.md)                       | GET    | Search for clans                    |
| [getClan](./getclan.md)                 | GET    | Get clan information                |
| [getClanInfoFull](./getclaninfofull.md) | GET    | Get detailed clan info with members |
| [update](./update.md)                   | POST   | Update clan settings                |
