# Achievement Endpoints

Endpoints for player achievements and statistics synchronization.

> **Authentication:** All endpoints in this section require Steam authentication.

## Endpoints

| Endpoint                                                  | Method | Description                        |
| --------------------------------------------------------- | ------ | ---------------------------------- |
| [getAchievements](./getachievements.md)                   | GET    | Get player's unlocked achievements |
| [getAvailableAchievements](./getavailableachievements.md) | GET    | Get all available achievements     |
| [syncStats](./syncstats.md)                               | POST   | Sync achievement statistics        |
