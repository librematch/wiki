# Chat Endpoints

Endpoints for in-game chat functionality and messaging.

> **Authentication:** All endpoints in this section require Steam authentication.
>
> **Warning:** Accessing chat channels may be monitored. Use sparingly.

## Endpoints

| Endpoint                                          | Method | Description                 |
| ------------------------------------------------- | ------ | --------------------------- |
| [deleteOfflineMessage](./deleteofflinemessage.md) | POST   | Delete an offline message   |
| [getChatChannels](./getchatchannels.md)           | GET    | Get available chat channels |
| [getOfflineMessages](./getofflinemessages.md)     | GET    | Retrieve offline messages   |
