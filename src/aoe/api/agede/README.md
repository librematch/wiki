# Age of Empires: Definitive Edition API

Public API endpoints for Age of Empires: Definitive Edition match history and statistics on ageofempires.com.

## Endpoints

| Endpoint                              | Method | Description                      |
| ------------------------------------- | ------ | -------------------------------- |
| [getCampaign](./getcampaign.md)       | GET    | Get campaign statistics          |
| [getMPFull](./getmpfull.md)           | GET    | Get full multiplayer statistics  |
| [getMPMatchList](./getmpmatchlist.md) | GET    | Get multiplayer match list       |
| [getSPFull](./getspfull.md)           | GET    | Get full singleplayer statistics |
| [getSPMatchList](./getspmatchlist.md) | GET    | Get singleplayer match list      |
