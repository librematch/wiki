# Age of Empires III: Definitive Edition API

Public API endpoints for Age of Empires III: Definitive Edition leaderboards on ageofempires.com.

## Endpoints

| Endpoint                                  | Method | Description                             |
| ----------------------------------------- | ------ | --------------------------------------- |
| [leaderboard](./leaderboard.md)           | GET    | Get leaderboard rankings                |
| [eventLeaderboard](./eventleaderboard.md) | GET    | Get event-specific leaderboard rankings |
