# Age of Empires IV API

Public API endpoints for Age of Empires IV leaderboards, events, and match details on ageofempires.com.

## Endpoints

| Endpoint                                  | Method | Description                             |
| ----------------------------------------- | ------ | --------------------------------------- |
| [leaderboard](./leaderboard.md)           | GET    | Get leaderboard rankings                |
| [eventLeaderboard](./eventleaderboard.md) | GET    | Get event-specific leaderboard rankings |
| [eventList](./eventlist.md)               | GET    | Get list of available events            |
| [getMatchDetail](./getmatchdetail.md)     | GET    | Get detailed match information          |
